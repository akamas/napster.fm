Saya alias dalam talian adalah buu700, tetapi saya sering pergi oleh Ryan Lester IRL. Saya seorang pelajar di Carnegie Mellon, kini mengambil beberapa tahun bekerja di SpaceX dan melakukan barangan yang lain. Dalam masa saya percuma, saya menyumbang kepada w3sec.

Saya ingin membuat pertanyaan kecil anda / spam anda / meminta anda keluar pada tarikh / memberikan anda wang / membawa anda ke Mahkamah / menghantar komen. Bagaimana saya boleh menghubungi anda?

Pasti, tiada masalah; Tatal ke bawah hanya untuk maklumat hubungi saya.

Saya perlu apa-apa lebih baik lakukan dengan hidup saya; Bagaimana saya boleh menyumbang kepada Peer.fm?

Seperti yang dinyatakan di halaman rumah, segala-galanya dibangunkan di buka di GitHub; mana-mana tangan tambahan akan cemerlang. E-mel saya atau sesuatu sebelum menghantar sebarang permintaan tarik, tetapi sila jangan teragak-agak untuk log masuk mana-mana laporan pepijat atau permintaan ciri dalam isu-isu sistem ini.

Jika saya perlu memutuskan untuk menyumbang, khusus Apakah teknikal mengenai Peer.fm saya harus sedar?

1. untuk menguruskan semua data kami, saya menggunakan kedai hebat data masa sebenar "serverless" Firebase (YC S11). Akibatnya, selain satu set kaedah-kaedah pengesahan Firebase, ini adalah tulen-klien Kejuruteraan; Gred-A penggodam JS sahaja.

2. Selain daripada Firebase SDK dan YouTube API, Perpustakaan Utama dan rangka kerja yang saya menggunakan adalah: Penutupan Google, AngularJS, Gula, dan Bootstrap (mereka semua cemerlang).

3. Jika anda sedang digunakan untuk terabas DOM dengan jQuery, ClosureQuery berasaskan penutupan saya mungkin kelihatan agak biasa.

Apa khusus anda boleh menggunakan bantuan?

Untuk preface, saya terbuka kepada apa-apa sumbangan yang dicadangkan untuk projek tersebut. Bahawa yang berkata:

1. ujian automatik: Peer.fm kini mempunyai liputan ujian sifar dan saya bercadang untuk menulis apa-apa diri saya di masa hadapan; ini boleh menjadi satu projek bagi orang lain untuk mengendalikan.

2. karya seni: saya buat masa ini saya meminjam logo Akonadi untuk favicon Peer.fm (tiada kepentingan khas; KDE's hanya menggerunkan).

3. pengguna dokumentasi: Tiada buat masa ini.
