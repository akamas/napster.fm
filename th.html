เย็น ดังนั้น ทำไมไม่ Peer.fm คลับคล้าย Windows 8

Windows 8 นำความคิดบางอย่างที่น่าสนใจและ UX เกือบปาน แต่ส่วนใหญ่แค่คิดแบบเมโทรใหม่ / แนวโน้มแบนดูเย็น

ยัง ฉันชอบความคิดของ reimagining Napster –เดิม Windows 95 เดสก์ท็อปอินเทอร์เฟซแบบ – เป็น Windows 8-ish ทันสมัยเว็บโปรแกรมประยุกต์

ที่พูดของเดสก์ท็อป มีปพลิเคชันดั้งเดิม และโทรศัพท์มือถือของฉัน เมื่อสามารถทำงานนี้ในเครื่องปิ้งขนมปังของฉัน

จาก ฉันมีความคิดบางอย่างว่า ถ้าคุณจริง ๆ ดูแล เพียง ยิงฉันอีเมล์ และฉันจะช่วยให้คุณทราบทันทีที่มีการปรับปรุง

คุณเป็นใคร

นามแฝงของฉันออนไลน์คือ buu700 แต่มักไป โดย Ryan รอนโต้ IRL ผมเรียนที่เมลลอนคาร์เนกี ในปัจจุบันมีกี่ปีปิด SpaceX และทำสิ่งอื่น ๆ ในเวลาว่างของฉัน ฉันนำไป w3sec

อยากจะทำการสอบถามขนาดเล็กของคุณสแปมคุณ/ ขอให้คุณออกในวัน/ ให้เงิน/ ไปศาล / ส่งข้อคิดเห็น วิธีสามารถฉันติดต่อคุณ

แน่ ไม่มีปัญหา เลื่อนไปด้านล่าง เพียงสำหรับข้อมูลที่ติดต่อของฉัน

ไม่มีอะไรดีขึ้นกับชีวิต ฉันจะสนับสนุนการ Peer.fm

ดังกล่าวบนโฮมเพจ ทุกอย่างจะพัฒนาในเปิดใน GitHub มือใด ๆ เพิ่มเติมจะสดใส อีเมล์ผมหรือบางสิ่งบางอย่างก่อนที่จะส่งคำขอใด ๆ ดึง แต่โปรดอย่าลังเลที่จะเข้าสู่รายงานข้อผิดพลาดใด ๆ หรือเตือนใน ปัญหาระบบ

ถ้าฉันตัดสินใจจะมีส่วนร่วม ประเภทอะไรทางเทคนิคเกี่ยวกับ Peer.fm ฉันควรทราบ

1. การจัดการข้อมูลของเราทั้งหมด ฉันใช้เก็บข้อมูล "serverless" แบบเรียลไทม์อัน Firebase (YC S11) ผล นอกเหนือจากชุดของกฎการตรวจสอบ Firebase นี้เป็นวิศวกรรมศาสตร์บริสุทธิ์ของไคลเอ็นต์ เกรด A JS กเกอร์เท่านั้น

2. จาก Firebase SDK และ YouTube API ไลบรารีที่สำคัญและกรอบที่ผมใช้อยู่: ปิด Google, AngularJS, น้ำตาล และ Bootstrap (พวกเขากำลังสดใสทั้งหมด)

3. ถ้าคุณกำลังใช้เพื่อข้ามสิ่งกีดขวางโดมกับ jQuery, ClosureQuery ของฉันตามปิดอาจดูคุ้นเคยเล็กน้อย

สิ่งเฉพาะสามารถคุณใช้ความช่วยเหลือหรือไม่

การนำ ฉันเปิดเงินสมทบโครงการต่าง ๆ เสนอ ที่ถูกกล่าวว่า:

1. การทดสอบอัตโนมัติ: Peer.fm ปัจจุบันมีศูนย์ครอบคลุม และผมแผนเขียนใด ๆ ตัวเองในอนาคต นี้อาจเป็นโครงการสำหรับบุคคลอื่นในการจัดการ

2. งานศิลปะ: ฉันกำลังขณะนี้ยืมโลโก้ Akonadi สำหรับ Peer.fm favicon (พิเศษความสำคัญ KDE ของน่ากลัวเพียง)

3. ผู้ใช้เอกสาร: มีไม่มีในขณะนี้

โดยวิธี เว็บไซต์นี้ทั้งหมดได้รับโดยอัตโนมัติแปลจากอังกฤษ ถ้าสิ่งที่ฉันได้เขียนเสียงบ้า หรือไม่เข้าใจ ก็คงทำไม การใช้ Peer.fm ในอังกฤษต้นฉบับ คลิกที่นี่
